# Расширения для поддержки синтаксиса 1С в VS Code / Cursor

## Рекомендуемое расширение файла

Для файлов с кодом на языке 1С используйте расширение **`.bsl`** (Business Scripting Language).

Пример: `Code.bsl`, `МодульОбъекта.bsl`, `ОбработчикКоманды.bsl`

## Установка расширения Language 1C (BSL)

### Способ 1: Через панель расширений

1. Откройте Cursor (или VS Code)
2. Нажмите `Ctrl+Shift+X` (или `Cmd+Shift+X` на Mac) для открытия панели расширений
3. В строке поиска введите: `BSL` или `1C`
4. Найдите расширение **"Language 1C (BSL)"** от разработчика 1c-syntax
5. Нажмите кнопку **"Установить"**

### Способ 2: Через командную палитру

1. Нажмите `Ctrl+Shift+P` (или `Cmd+Shift+P` на Mac)
2. Введите: `Extensions: Install Extensions`
3. В поиске введите: `BSL`
4. Выберите **"Language 1C (BSL)"** и установите

### Способ 3: Через ID расширения

1. Откройте терминал в Cursor
2. Выполните команду:
   ```bash
   code --install-extension 1c-syntax.language-1c-bsl
   ```

## Популярные расширения для 1С

### 1. Language 1C (BSL)
- **ID**: `1c-syntax.language-1c-bsl`
- **Функции**:
  - Подсветка синтаксиса языка 1С
  - Автодополнение кода
  - Поддержка языка запросов
  - Форматирование кода

### 2. 1C: Enterprise Development Tools (EDT)
- Это полноценная IDE на базе Eclipse
- Рекомендуется для серьезной разработки конфигураций
- Скачать: https://releases.1c.ru/project/DevelopmentTools10

## Настройка файлов

### Кодировка файлов
- Используйте кодировку **UTF-8 без BOM**
- В Cursor/VS Code: `File` → `Preferences` → `Settings` → поиск "encoding"
- Установите: `"files.encoding": "utf8"`

### Ассоциация файлов
Расширение `.bsl` автоматически распознается после установки расширения.

Если нужно добавить другие расширения:
1. Откройте настройки (`Ctrl+,`)
2. Найдите `files.associations`
3. Добавьте:
   ```json
   {
     "files.associations": {
       "*.bsl": "bsl",
       "*.os": "bsl"
     }
   }
   ```

## Проверка установки

1. Создайте файл `test.bsl`
2. Введите код:
   ```bsl
   Переменная = 10;
   Сообщить("Привет, 1С!");
   ```
3. Если синтаксис подсвечивается цветом - расширение работает!

## Форматы файлов 1С

- **`.bsl`** - файлы с кодом на языке 1С (Business Scripting Language)
- **`.cf`** - файл конфигурации
- **`.cfe`** - файл расширения конфигурации
- **`.epf`** - файл внешней обработки
- **`.erf`** - файл внешнего отчета
- **`.dt`** - файл выгрузки информационной базы
