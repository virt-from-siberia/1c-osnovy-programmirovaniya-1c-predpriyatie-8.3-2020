# Основы программирования 1С:Предприятие 8.3

## О проекте
Учебная конфигурация для изучения основ программирования в 1С.

## Структура репозитория

```
├── Base/               # База данных (игнорируется git)
├── ОП-Начальная.dt     # Выгрузка конфигурации
├── Code.bsl            # Примеры кода на языке 1С
├── VSCODE_EXTENSIONS.md # Инструкция по установке расширений
└── TIME.txt            # Служебный файл
```

## Как работать с конфигурацией

### Вариант 1: Загрузка из .dt файла (текущий способ)

1. Создайте новую информационную базу в 1С
2. Выполните восстановление из файла `ОП-Начальная.dt`

```bash
"C:\Program Files\1cv8\8.3\bin\1cv8.exe" DESIGNER /F ".\Base" /RestoreIB "ОП-Начальная.dt"
```

### Вариант 2: Выгрузка в XML (рекомендуется для совместной разработки)

Чтобы видеть изменения в коде построчно:

```bash
# Выгрузить конфигурацию в XML
"C:\Program Files\1cv8\8.3\bin\1cv8.exe" DESIGNER /F ".\Base" /DumpConfigToFiles ".\src" -format Hierarchical

# После внесения изменений - обновить .dt файл
"C:\Program Files\1cv8\8.3\bin\1cv8.exe" DESIGNER /F ".\Base" /DumpIB "ОП-Начальная.dt"
```

## Типичный workflow разработчика 1С с git

1. **Получить изменения:**
   ```bash
   git pull
   ```

2. **Загрузить конфигурацию в базу**

3. **Внести изменения в конфигураторе**

4. **Выгрузить конфигурацию:**
   - В .dt файл (простой способ)
   - Или в XML (профессиональный способ)

5. **Зафиксировать изменения:**
   ```bash
   git add .
   git commit -m "Добавил обработку расчета зарплаты"
   git push
   ```

## Работа с кодом в Cursor/VS Code

### Установка расширения для подсветки синтаксиса

Для работы с кодом 1С в Cursor/VS Code установите расширение **Language 1C (BSL)**:

1. Нажмите `Ctrl+Shift+X` для открытия панели расширений
2. Найдите и установите: **"Language 1C (BSL)"**
3. Файлы с кодом должны иметь расширение **`.bsl`**

Подробная инструкция: см. файл [VSCODE_EXTENSIONS.md](VSCODE_EXTENSIONS.md)

### Примеры кода

В файле `Code.bsl` содержатся примеры кода на языке 1С для изучения основ программирования.
